# Job Application System 🚀

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Build Status](https://img.shields.io/badge/build-passing-brightgreen.svg)

Hệ thống quản lý đơn ứng tuyển việc làm, giúp người dùng dễ dàng tìm kiếm công việc và nhà tuyển dụng quản lý hồ sơ ứng viên một cách hiệu quả.

---

## 📌 Tính năng chính

### 👤 Dành cho Ứng viên
*   **Tìm kiếm:** Tìm kiếm việc làm theo từ khóa, địa điểm và danh mục.
*   **Hồ sơ:** Tạo và quản lý hồ sơ cá nhân (CV/Resume).
*   **Ứng tuyển:** Nộp đơn ứng tuyển trực tuyến nhanh chóng.
*   **Theo dõi:** Cập nhật trạng thái đơn hàng (Đã nộp, Đang xem xét, Phỏng vấn, Từ chối).

### 🏢 Dành cho Nhà tuyển dụng
*   **Đăng tin:** Tạo và quản lý các bài đăng tuyển dụng mới.
*   **Quản lý ứng viên:** Xem danh sách ứng viên cho từng vị trí cụ thể.
*   **Lọc thông minh:** Phân loại và lọc hồ sơ ứng viên hiệu quả.
*   **Thông báo:** Gửi phản hồi hoặc thông báo tự động cho ứng viên qua email.

## 🚀 Cài đặt và Chạy thử

### 📋 Yêu cầu hệ thống
*   Node.js (v14+) hoặc Docker.
*   Database tương ứng đã được cài đặt và cấu hình.

### ⚙️ Các bước cài đặt

1.  **Clone repository:**
    ```bash
    git clone [https://github.com/LuongHaMinhVy/Job-Application.git](https://github.com/LuongHaMinhVy/Job-Application.git)
    cd Job-Application
    ```

2.  **Cài đặt dependencies:**
    *   **Cho Backend:**
        ```bash
        cd backend
        npm install
        ```
    *   **Cho Frontend:**
        ```bash
        cd ../frontend
        npm install
        ```

3.  **Cấu hình môi trường:**
    Tạo file `.env` từ file `.env.example` và điền các thông số Database, API Key.

4.  **Khởi chạy ứng dụng:**
    ```bash
    npm run dev
    ```
